## How to follow along?

Create a brand new **Flutter** project using the following command to follow along.

```bash
flutter create --org com.tutorial --description 'your_description' flutter_tutorial -i swift -a kotlin
```

## Folder Structure Used

- lib/

    - services
  - ui
